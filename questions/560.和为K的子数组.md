**示例 1：**

**输入：**nums = [1,1,1], k = 2
**输出：**2

**示例 2：**

**输入：**nums = [1,2,3], k = 3
**输出：**2

**提示：**

- `1 <= nums.length <= 2 * 104`
- `-1000 <= nums[i] <= 1000`
- `-107 <= k <= 107`
