# 选中的参考答案

官方题解3：牛顿迭代法
```c++
class Solution {
public:
    int mySqrt(int x) {
        if (x == 0) {
            return 0;
        }
        double C = x, x0 = x;
        while (true) {
            double xi = 0.5 * (x0 + C / x0);
            if (fabs(x0 - xi) < 1e-7) {
                break;
            }
            x0 = xi;
        }
        return int(x0);
    }
};
```

# 评论

解答也是官方题解解法之一，但牛顿迭代法用得多，而且效率也不慢。
