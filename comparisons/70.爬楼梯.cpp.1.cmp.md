# 选中的参考答案

``` c++
class Solution {
public:
	int climbStairs(int n) {
		int p = 0, q = 0, r = 1;
		for (int i = 1; i <= n; ++i) {
			p = q; q = r; r = p + q;
		}
		return r;
	}
};
```

# 评论

动态规划，暂时来看，和循环不变式原理如出一辙：

- 定义一种数值关系；
- 循环不变式保持；
- 循环结束，从这个数值关系的一部分中提取有用信息。

循环到任一进度，数值关系的取值就是一个状态。

状态压缩，就是丢弃对循环保持和循环结束时无用的状态。
